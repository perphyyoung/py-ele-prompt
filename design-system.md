# 设计规范

> 本文档定义项目的 UI/UX 设计标准，所有界面实现必须遵循此规范。

## 1. 通用设计原则

### 1.1 对齐原则

- **同级按钮对齐且大小一致**
- 表单标签与输入框左对齐
- 卡片内元素保持统一间距

### 1.2 交互反馈

- **删除操作需用户确认**（弹窗确认）
- 按钮悬停有明显视觉反馈
- 操作成功/失败有 Toast 提示

### 1.3 数据展示

- **数据目录禁止硬编码**，使用配置项
- 空状态显示友好提示
- 加载状态显示骨架屏或加载动画

## 2. 收藏按钮

### 2.1 规格

| 属性 | 值 |
| --- | --- |
| 尺寸 | 28px × 28px |
| 形状 | 圆形（border-radius: 50%） |
| 边框 | 1px solid var(--border-color) |
| 背景 | var(--card-bg) |
| 图标颜色 | var(--text-secondary) |

### 2.2 状态样式

| 状态 | 背景 | 边框 | 图标颜色 |
| --- | --- | --- | --- |
| 默认 | var(--card-bg) | var(--border-color) | var(--text-secondary) |
| 悬停 | rgba(245, 158, 11, 0.1) | rgba(245, 158, 11, 0.3) | #d97706 |
| 已收藏 | linear-gradient(135deg, #f59e0b, #fbbf24) | transparent | white |
| 已收藏悬停 | linear-gradient(135deg, #d97706, #f59e0b) | transparent | white |

### 2.3 使用场景

| 场景 | 显示逻辑 |
| --- | --- |
| 卡片内（网格视图） | 悬停卡片时显示，已收藏的始终显示 |
| 列表项（列表视图） | 始终显示 |
| 详情页 | 始终显示 |
| 编辑页 | 始终显示 |

### 2.4 CSS 类

- `.favorite-btn` - 基础样式（28px 圆形）

### 2.5 代码示例

```html
<!-- 默认状态 -->
<button class="favorite-btn" title="收藏"></button>

<!-- 已收藏状态 -->
<button class="favorite-btn active" title="取消收藏"></button>
```

```css
/* 基础样式 */
.favorite-btn {
  width: 28px;
  height: 28px;
  border-radius: 50%;
  border: 1px solid var(--border-color);
  background: var(--card-bg);
  color: var(--text-secondary);
  cursor: pointer;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: all 0.2s ease;
}

/* 悬停状态 */
.favorite-btn:hover {
  background: rgba(245, 158, 11, 0.1);
  border-color: rgba(245, 158, 11, 0.3);
  color: #d97706;
}

/* 已收藏状态 */
.favorite-btn.active {
  background: linear-gradient(135deg, #f59e0b 0%, #fbbf24 100%);
  border-color: transparent;
  color: white;
}

.favorite-btn.active:hover {
  background: linear-gradient(135deg, #d97706 0%, #f59e0b 100%);
}
```

## 3. 按钮规范

### 3.1 按钮类型

| 类型 | CSS 类 | 用途 |
| --- | --- | --- |
| 主要按钮 | `btn btn-primary` | 主要操作（保存、确认） |
| 次要按钮 | `btn btn-secondary` | 次要操作（取消、返回） |
| 危险按钮 | `btn btn-danger` | 删除、清空等危险操作 |
| 模态框按钮 | `btn-modal` | 模态框底部按钮 |

### 3.2 按钮原则

- **同级按钮对齐且大小一致**
- 一个区域最多一个主要按钮
- 危险操作需二次确认

## 4. 表单规范

### 4.1 必需字段标记

- 使用 `required` 类标记表单组
- 标签后添加红色星号：`<span class="required-mark">*</span>`
- 必需字段标签显示为紫色

### 4.2 表单布局

```html
<div class="form-group required">
  <label>
    字段名
    <span class="required-mark">*</span>
  </label>
  <input type="text" class="form-control">
</div>
```

## 5. 模态框规范

### 5.1 模态框尺寸

| 尺寸 | CSS 类 | 用途 |
| --- | --- | --- |
| 大模态框 | `modal-content-large` | 详情页、编辑页 |
| 中模态框 | `modal-content-medium` | 确认对话框 |
| 小模态框 | `modal-content-small` | 简单提示 |

### 5.2 模态框结构

```text
+------------------+
| 标题栏        [X] |
+------------------+
|                  |
|   内容区域        |
|                  |
+------------------+
| [取消]  [确认]   |
+------------------+
```

## 6. 卡片规范

### 6.1 卡片结构

- **卡片元素**: 首图背景/缩略图、标题、内容预览、标签、收藏按钮、操作按钮
- 卡片悬停显示操作按钮（已收藏状态始终显示）
- 卡片支持网格视图和列表视图

### 6.2 卡片视图

| 视图 | 特点 |
| --- | --- |
| 网格视图 | 卡片网格布局，显示首图背景 |
| 列表视图 | 横向列表项，紧凑显示 |

## 7. 布局结构

### 7.1 整体布局

```text
+------------------------------------------+
|  标题栏 (Prompt Manager)                 |
+----------+-------------------------------+
|          |                               |
|  侧边栏  |        内容区域               |
|  (导航)  |                               |
|          |  - 提示词管理                  |
|          |  - 图像管理                    |
|          |  - 设置                        |
|          |                               |
+----------+-------------------------------+
```

### 7.2 内容区域结构

- **工具栏**: 新建、搜索、排序、视图切换、标签筛选
- **内容区**: 卡片网格或列表

## 11. 标签规范

### 11.1 标签样式

- 标签显示为圆角小标签
- 特殊标签（收藏、安全评级等）居中展示
- 特殊标签与普通标签用竖线分隔

### 11.2 标签筛选

- 支持单选/多选标签组合筛选
- 单选组内的标签互斥选择
- 标签按使用数量和名称排序

## 12. 界面状态

### 12.1 空状态

- 显示友好图标和提示文字
- 提供快捷操作入口（如"新建"按钮）

### 12.2 加载状态

- 显示骨架屏或加载动画
- 避免界面闪烁

### 12.3 错误状态

- 显示错误提示
- 提供重试或返回入口
