# 数据模型

> 本文档描述 Prompt Manager 的数据库表结构和数据关系。

---

## 1. 核心表

### 1.1 提示词表 (prompts)

| 字段 | 类型 | 说明 |
| ------ | ------ | ------ |
| id | TEXT | 主键 |
| title | TEXT | 标题 |
| content | TEXT | 内容 |
| content_translate | TEXT | 翻译 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
| is_deleted | INTEGER | 是否删除 |
| deleted_at | DATETIME | 删除时间 |
| is_favorite | INTEGER | 是否收藏 |
| is_safe | INTEGER | 安全评级 |
| note | TEXT | 备注 |

### 1.2 图像表 (images)

| 字段 | 类型 | 说明 |
| ------ | ------ | ------ |
| id | TEXT | 主键 |
| file_name | TEXT | 文件名 |
| stored_name | TEXT | 存储名 |
| relative_path | TEXT | 相对路径 |
| thumbnail_path | TEXT | 缩略图路径 |
| width | INTEGER | 宽度 |
| height | INTEGER | 高度 |
| file_size | INTEGER | 文件大小 |
| created_at | DATETIME | 创建时间 |
| is_deleted | INTEGER | 是否删除 |
| deleted_at | DATETIME | 删除时间 |
| is_favorite | INTEGER | 是否收藏 |
| is_safe | INTEGER | 安全评级 |
| note | TEXT | 备注 |
| gen_params | TEXT | 生成参数 (JSON) |

## 2. 关联表

### 2.1 提示词 - 图像关联

**prompt_image_relations** 提示词 - 图像关联表

| 字段 | 类型 | 说明 |
| ------ | ------ | ------ |
| prompt_id | TEXT | 提示词 ID |
| image_id | TEXT | 图像 ID |
| created_at | DATETIME | 创建时间 |

### 2.2 标签表

**prompt_tags** - 提示词标签
**image_tags** - 图像标签

| 字段 | 类型 | 说明 |
| ------ | ------ | ------ |
| id | TEXT | 主键 |
| name | TEXT | 标签名称 |
| type | TEXT | 标签类型（prompt/image） |
| created_at | DATETIME | 创建时间 |

### 2.3 标签关联表

**prompt_tag_relations** - 提示词 - 标签关联
**image_tag_relations** - 图像 - 标签关联

| 字段 | 类型 | 说明 |
| ------ | ------ | ------ |
| tag_id | TEXT | 标签 ID |
| target_id | TEXT | 目标 ID（prompt_id 或 image_id） |
| created_at | DATETIME | 创建时间 |

---

## 3. 数据关系

``` bash
prompts (提示词)
├── prompt_image_relations (关联图像)
│   └── images (图像)
└── prompt_tag_relations (关联标签)
    └── prompt_tags (标签)

images (图像)
├── prompt_image_relations (关联提示词)
│   └── prompts (提示词)
└── image_tag_relations (关联标签)
    └── image_tags (标签)
```

## 5. 软删除机制

所有表都使用软删除机制：

- **is_deleted = 0** - 正常数据
- **is_deleted = 1** - 已删除数据（回收站）
- **deleted_at** - 删除时间戳

### 查询示例

```sql
-- 查询正常数据
SELECT * FROM prompts WHERE is_deleted = 0;

-- 查询回收站数据
SELECT * FROM prompts WHERE is_deleted = 1;

-- 彻底删除
DELETE FROM prompts WHERE is_deleted = 1;
```

## 8. 关联数据结构

### 8.1 promptRefs - 图像关联的提示词引用

`promptRefs` 是图像关联的提示词引用数组，用于前端展示和维护关联关系。

```javascript
promptRefs: [
  {
    promptId: String,            // 提示词 ID
    promptTitle: String,         // 提示词标题
    promptContent: String,       // 提示词内容
    promptContentTranslate: String, // 提示词翻译（可选）
    promptNote: String,          // 提示词备注（可选）
  }
]
```

**说明**：

- 使用 `promptId` 而非 `id`，避免与图像自身的 `id` 字段混淆
- 主要用于图像详情界面显示关联的提示词列表
- 解除关联时使用 `p.promptId` 进行比较

### 8.2 prompt.images - 提示词关联的图像

提示词关联的图像存储在 `prompt.images` 中，是完整的图像对象数组：

```javascript
prompt.images: [
  {
    id: String,                 // 图像 ID
    fileName: String,           // 文件名
    storedName: String,         // 存储名
    relativePath: String,       // 相对路径
    thumbnailPath: String,       // 缩略图路径
    width: Number,              // 宽度
    height: Number,             // 高度
    fileSize: Number,           // 文件大小
    isFavorite: Boolean,        // 是否收藏
    isSafe: Number,             // 安全评级 (0/1)
    isDeleted: Boolean,         // 是否删除
    note: String,               // 备注
    createdAt: String,           // 创建时间
    updatedAt: String,           // 更新时间
    tags: Array<String>,        // 标签数组
  }
]
```

**说明**：

- 与 `promptRefs` 不同，`prompt.images` 存储完整的图像对象
- 用于提示词详情界面显示关联的图像列表
