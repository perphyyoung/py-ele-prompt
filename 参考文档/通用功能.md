## 1. 标签管理系统

### 适用范围

- ✅ 提示词标签
- ✅ 图像标签

### 功能列表

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 标签添加 | 支持为对象添加多个标签 | P0 |
| 标签筛选 | 支持单选/多选标签组合筛选（Shift 键多选） | P0 |
| 标签组管理 | 创建标签组，设置组类型（单选/多选），按优先级排序，卡片显示排序数值 | P0 |
| 标签拖拽 | 拖拽标签分配到不同组，支持从标签筛选区拖拽标签到卡片快速添加 | P1 |
| 单选组冲突处理 | 拖拽单选组标签到已有同组标签的卡片时，提示用户选择保留哪个 | P2 |
| 标签排序 | 按使用数量和名称排序 | P1 |
| 特殊标签 | 收藏、安全评级等特殊标签显示，居中展示 | P1 |
| 标签分隔 | 特殊标签与普通标签用竖线分隔 | P2 |
| 标签筛选收起 | 标签筛选区域支持收起/展开，收起时显示优先级最高的标签组及其标签（过滤计数为 0 的标签） | P2 |
| 单选组限制 | 单选组内的标签互斥选择，标题栏中的单选组标签同样遵循限制 | P2 |
| 首位组标识 | 标签管理界面首位组卡片标题栏显示"首位组"标识 | P2 |
| 固定到首位 | 标签组支持右键菜单固定到首位功能 | P2 |
| 特殊标签校验 | 新建/重命名标签时校验特殊标签，仅提醒不清空输入框 | P2 |

### 特殊标签类型

#### 提示词特殊标签

- **收藏** - 标记为收藏的提示词
- **多图** - 关联多张图像的提示词
- **无图** - 未关联图像的提示词
- **安全评级** - Safe/NSFW 标识

#### 图像特殊标签

- **收藏** - 标记为收藏的图像
- **未引** - 未被提示词引用的图像
- **多引** - 被多个提示词引用的图像
- **安全评级** - Safe/NSFW 标识

- 显示缩略图、标题/文件名、标签

- 适合快速浏览大量项目

#### 批量操作

### 删除流程

```

  ↓
用户恢复 → 标记 is_deleted=0 → 回到原列表
  ↓
用户彻底删除 → 从数据库删除 → 不可恢复
```

## 7. 自动保存系统

### 适用范围

- ✅ 提示词编辑
- ✅ 图像详情编辑
- ✅ 所有 textarea 输入框

### 功能列表

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 防抖自动保存 | 输入后 800ms 自动保存，避免频繁更新 | P0 |
| 字段级变化追踪 | 只保存变化的字段，避免不必要的数据库更新 | P0 |
| 自动调整高度 | 所有 textarea 输入框根据内容自动调整高度 | P1 |

### 防抖机制

``` bash
用户输入 → 触发 input 事件 → 启动 800ms 定时器
  ↓
定时器到期 → 检查是否有变化 → 保存到数据库
  ↓
用户再次输入 → 清除旧定时器 → 启动新定时器
```

### 字段级变化追踪

- 记录每个字段的原始值
- 只保存变化的字段
- 减少数据库更新次数

### 自动调整高度

- 使用业界通用方案
- 重置 height 为 auto 后设置为 scrollHeight
- requestAnimationFrame 确保 DOM 更新后计算
- 避免初始化时 scrollHeight 为 0

## 8. 安全评级系统

### 适用范围

- ✅ 提示词安全评级
- ✅ 图像安全评级

### 功能列表

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 安全评级设置 | 支持设置 Safe/NSFW 评级 | P0 |
| 双向联动 | 提示词与关联图像的安全评级双向联动（事件驱动架构） | P1 |
| 视图模式控制 | Safe/NSFW 模式切换控制内容显示范围 | P1 |

### 双向联动机制

#### 联动规则

- **提示词 → 图像** - 修改提示词安全评级，联动更新所有关联图像
- **图像 → 提示词** - 修改图像安全评级，联动更新所有关联提示词
- **以最后操作为准** - 多对多关系时，最后操作的评级覆盖之前的

#### 技术实现

- **事件驱动架构** - 使用发布 - 订阅模式
- **SafeRatingService** - 集中管理联动逻辑
- **数据层解耦** - 数据库层只负责单个对象更新
- **UI 层同步** - 订阅事件自动更新界面

``` bash
用户操作 → SafeRatingService → 更新数据库 → 发布事件
  ↓
app.js 订阅事件 → 更新缓存 → 刷新 UI
```

### 特殊标签

- **Safe** - 安全内容（is_safe=1）
- **NSFW** - 不安全内容（is_safe=0）
