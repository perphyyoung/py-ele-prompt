# 特定功能

> 本文档只描述提示词和图像的**特定功能**。
>
> **通用功能**详见：[通用功能.md](通用功能.md)
> **数据模型**详见：[数据模型.md](数据模型.md)

---

## 1. 提示词特定功能

### 1.1 核心功能差异

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 字段 | 标题、内容、翻译、备注、图像关联 | P0 |
| 简化模式 | 只保留提示词内容和图像 | P1 |
| 搜索 | 支持内容和备注搜索 | P0 |
| 排序 | 按更新时间、创建时间、标题排序 | P1 |

### 1.2 图像关联（独有）

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 添加图像 | 支持上传本地图像或从图像管理选择 | P0 |
| 多图关联 | 一个提示词可关联多张图像 | P1 |
| 首图显示 | 卡片视图显示首图背景 | P1 |
| Hover 预览 | 卡片/列表项 hover 同时显示提示词内容和首图预览 | P2 |
| 图像标签显示 | 提示词编辑界面的图像缩略图底部叠加显示图像标签 | P2 |
| 安全评级联动 | 提示词安全评级与关联图像双向联动（事件驱动架构，SafeRatingService 集中管理） | P1 |

### 1.3 提示词特殊标签

- **多图** - 关联多张图像的提示词
- **无图** - 未关联图像的提示词

> 其他特殊标签（收藏、安全评级）详见 [通用功能.md#1-标签管理系统](通用功能.md#1-标签管理系统)。

## 2. 图像特定功能

### 2.1 核心功能差异

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 字段 | 文件名、备注、标签、尺寸、上传时间 | P0 |
| 格式支持 | JPG、PNG、GIF、WebP | P0 |
| 简化模式 | 只保留图像和提示词内容 | P1 |
| 搜索 | 支持文件名和备注搜索 | P0 |
| 排序 | 按更新时间、上传时间、文件名、尺寸排序 | P1 |

### 2.2 提示词关联（独有）

| 功能 | 描述 | 优先级 |
| ------ | ------ | -------- |
| 多引用管理 | 一张图像可关联多个提示词，图像详情显示所有关联提示词 | P1 |
| 解除关联 | 支持解除与提示词的关联，同步更新双方更新时间 | P1 |
| 未引用识别 | 识别未被提示词引用的图像，支持从图像详情添加提示词关联 | P1 |
| 提示词预览 | 鼠标悬停显示关联提示词内容 | P2 |
| 安全评级联动 | 图像安全评级与关联提示词双向联动（事件驱动架构，SafeRatingService 集中管理） | P1 |

### 2.3 图像特殊标签

- **未引** - 未被提示词引用的图像
- **多引** - 被多个提示词引用的图像

> 其他特殊标签（收藏、安全评级）详见 [通用功能.md#1-标签管理系统](通用功能.md#1-标签管理系统)。
