## 使用建议

### 新开发者

1. 先阅读 [README.md](../README.md) 了解项目概况
2. 查看 [特定功能.md](特定功能.md) 了解业务逻辑
3. 参考 [数据模型.md](数据模型.md) 理解数据结构
4. 遵循 [设计规范.md](设计规范.md) 开发界面

### 功能开发

1. 查看 [通用功能.md](通用功能.md) 了解通用功能
2. 查看 [特定功能.md](特定功能.md) 了解特定功能
3. 参考 [数据模型.md](数据模型.md) 设计数据库操作

### 界面开发

1. 查看 [设计规范.md](设计规范.md) 了解 UI 规范
2. 查看 [通用功能.md](通用功能.md) 了解交互逻辑
3. 参考 [特定功能.md](特定功能.md) 了解业务场景

## 维护说明

### 文档更新原则

1. **通用功能** 变更 → 更新 [通用功能.md](通用功能.md)
2. **特定功能** 变更 → 更新 [特定功能.md](特定功能.md)
3. **数据库** 变更 → 更新 [数据模型.md](数据模型.md)
4. **UI 样式** 变更 → 更新 [设计规范.md](设计规范.md)

### 交叉引用更新

修改文档后，检查并更新：

- 文档顶部的"详见"引用
- 文档底部的"文档引用"章节
- README.md 中的相关引用
