## 内容概要

这个 skill 包含 **12 个核心模块**：

### 1. 📊 索引设计（P0 优先级）
- 何时创建索引
- 索引创建模式（单列、复合、部分索引）
- 索引选择策略
- 分析索引使用（EXPLAIN QUERY PLAN）

### 2. ⚡ 查询优化
- 避免 SELECT *
- 使用 EXISTS 代替 IN
- 避免 N+1 查询
- UNION ALL vs UNION
- 优化 LIKE 查询
- 使用 CTE（公共表表达式）

### 3. 🔒 事务管理
- 批量插入（10-100x 更快）
- 预编译语句 + 事务
- 嵌套事务（Savepoints）
- 乐观锁实现

### 4. 🔌 连接管理
- 单连接模式（Electron 应用）
- 连接池模式（高并发场景）
- 连接复用策略

### 5. ⚙️ PRAGMA 配置
- journal_mode = WAL
- synchronous = NORMAL
- cache_size = -64000 (64MB)
- foreign_keys = ON
- temp_store = MEMORY
- mmap_size = 256MB

### 6. 🏗️ 模式设计
- 规范化 vs 反规范化
- 索引策略
- 触发器更新反规范化字段

### 7. 📈 性能监控
- 查询分析
- 慢查询日志
- 表统计信息（ANALYZE）
- 数据库维护（VACUUM）

### 8. ❌ 常见反模式
- 缺失索引
- 在索引列上使用函数
- OR 条件阻止索引使用
- SQL 注入风险
- 大事务持有锁太久

### 9. 📝 代码生成指南
- 10 条核心编码原则

### 10. 🎯 主动应用模式
- 自动检测 SQLite 项目
- 代码审查检查清单

### 11. ✅ 性能检查清单
- 12 项部署前验证

### 12. 🔗 额外资源
- 官方文档链接

## 示例：自动应用

当你写这样的查询时：

```javascript
const prompts = await db.query(
  'SELECT * FROM prompts WHERE is_deleted = 0 ORDER BY updated_at DESC'
)
```

Skill 会自动建议：

1. ✅ 创建复合索引：`CREATE INDEX idx_prompts_deleted_updated ON prompts(is_deleted, updated_at DESC)`
2. ✅ 使用 EXPLAIN QUERY PLAN 验证索引使用
3. ✅ 只选择需要的列，而不是 SELECT *
