# Prompt Manager

本地提示词管理工具，基于 Electron 开发。

## 功能

### 提示词相关

- **提示词管理** - 创建、编辑、删除、搜索提示词，支持标题重复检测和覆盖，编辑时即时保存（回车/Ctrl+回车/失去焦点自动保存）
- **提示词标签** - 支持标签筛选和管理，Shift 键多选标签进行组合筛选
- **标签组管理** - 支持创建标签组，设置组类型（单选/多选），按优先级排序显示，卡片显示排序数值
- **标签拖拽** - 支持拖拽标签分配到不同组，支持从标签筛选区拖拽标签到卡片快速添加
- **标签排序** - 标签管理界面和标签筛选区域支持按使用数量和名称排序
- **特殊标签** - 标签管理界面显示特殊标签（收藏、多图、无图、安全评级），始终显示在搜索框上方，居中展示
- **标签分隔** - 标签筛选栏中，特殊标签与普通标签用竖线分隔
- **标签筛选收起** - 标签筛选区域支持收起/展开，收起时显示优先级最高的标签组及其标签（过滤计数为0的标签）
- **单选组限制** - 单选组内的标签互斥选择，标题栏中的单选组标签同样遵循限制
- **首位组标识** - 标签管理界面首位组卡片标题栏显示"首位组"标识
- **固定到首位** - 标签组支持右键菜单固定到首位功能
- **特殊标签校验** - 新建/重命名标签时校验特殊标签，仅提醒不清空输入框
- **收藏功能** - 提示词支持收藏/取消收藏，快速访问
- **排序功能** - 支持按更新时间、创建时间、标题排序
- **搜索功能** - 数据库层面的内容搜索，支持防抖优化，支持搜索提示词内容和备注
- **搜索清空** - 搜索框右侧提供清空按钮，快速清除搜索内容
- **视图切换** - 支持卡片视图、列表视图、紧凑视图三种模式切换
  - 卡片视图：显示首图背景、标题、内容预览
  - 列表视图：显示首图缩略图、标题、内容、标签
  - 紧凑视图：显示缩略图、标题、标签，行高更紧凑
- **Hover 预览** - 提示词卡片/列表项 hover 同时显示提示词内容和首图预览
- **列表多选** - 支持复选框选择、Ctrl+点击、Shift+范围选择、Ctrl+A 全选
- **批量操作** - 反选、批量收藏/取消收藏、批量添加标签、批量删除、Esc 取消选择，批量操作后自动更新标签计数
- **标签布局** - 文件名与标签同行显示，标签靠右排列
- **自动调整高度** - 所有 textarea 输入框根据内容自动调整高度
- **回收站** - 删除的提示词可恢复，卡片布局展示
- **导入导出** - 支持 JSON 格式备份
- **安全评级** - 支持设置提示词安全评级（safe/nsfw），与关联图像双向联动

### 图像相关

- **图像管理** - 为提示词添加本地图像，支持图像管理和详情查看
- **图像标签** - 支持标签筛选和管理，Shift 键多选标签进行组合筛选
- **标签组管理** - 支持创建标签组，设置组类型（单选/多选），按优先级排序显示，卡片显示排序数值
- **标签拖拽** - 支持拖拽标签分配到不同组，支持从标签筛选区拖拽标签到卡片快速添加
- **标签排序** - 标签管理界面和标签筛选区域支持按使用数量和名称排序
- **特殊标签** - 标签管理界面显示特殊标签（收藏、未引、多引、安全评级），始终显示在搜索框上方，居中展示
- **标签分隔** - 标签筛选栏中，特殊标签与普通标签用竖线分隔
- **标签筛选收起** - 标签筛选区域支持收起/展开，收起时显示优先级最高的标签组及其标签（过滤计数为0的标签）
- **单选组限制** - 单选组内的标签互斥选择，标题栏中的单选组标签同样遵循限制
- **首位组标识** - 标签管理界面首位组卡片标题栏显示"首位组"标识
- **固定到首位** - 标签组支持右键菜单固定到首位功能
- **特殊标签校验** - 新建/重命名标签时校验特殊标签，仅提醒不清空输入框
- **收藏功能** - 图像支持收藏/取消收藏，快速访问
- **排序功能** - 支持按更新时间、上传时间、文件名、尺寸排序
- **搜索功能** - 支持搜索图像文件名和备注
- **搜索清空** - 搜索框右侧提供清空按钮，快速清除搜索内容
- **视图切换** - 支持卡片视图、列表视图、紧凑视图三种模式切换
  - 卡片视图：显示缩略图、文件名、标签、操作按钮
  - 列表视图：显示缩略图、文件名、标签、提示词内容、备注
  - 紧凑视图：显示缩略图、文件名、标签，行高更紧凑
- **Hover 预览** - 图像卡片/列表项 hover 同时显示关联提示词内容和图像预览
- **列表多选** - 支持复选框选择、Ctrl+点击、Shift+范围选择、Ctrl+A 全选
- **批量操作** - 反选、批量收藏/取消收藏、批量添加标签、批量删除、Esc 取消选择，批量操作后自动更新标签计数
- **标签布局** - 文件名与标签同行显示，标签靠右排列
- **图像导航** - 图像详情页支持左右切换、键盘导航、全屏查看，导航按钮统一为圆形悬浮样式
- **图像提示词预览** - 鼠标悬停图像卡片显示关联提示词
- **提示词备注显示** - 图像详情界面显示关联提示词的备注
- **图像文件名编辑** - 图像详情界面支持编辑文件名，防抖自动保存（800ms），保存时显示状态（已保存/保存失败）
- **图像备注编辑** - 图像详情界面支持编辑备注，防抖自动保存（800ms），保存时显示状态
- **多引用管理** - 支持一张图像关联多个提示词，图像详情显示所有关联提示词，标签栏显示多引用筛选
- **解除关联** - 图像详情界面和提示词编辑界面支持解除与提示词的关联关系，解除时同步更新双方更新时间
- **未引用图像** - 识别和管理未被提示词引用的图像，标签栏显示未引用筛选，支持从图像详情添加提示词关联
- **回收站** - 删除的图像可恢复，卡片布局展示
- **安全评级** - 支持设置图像安全评级（safe/nsfw），与关联提示词双向联动
- **界面简化** - 图像上传界面只保留图像和提示词内容；新建提示词界面只保留提示词内容和图像
- **图像标签显示** - 提示词编辑界面的图像缩略图底部叠加显示图像标签

### 选择图像界面

- **独立排序** - 选择图像界面拥有独立的排序设置，与图像管理界面分开
- **搜索清空** - 搜索框右侧提供清空按钮，快速清除搜索内容

### 其他

- **主题切换** - 明亮/暗黑模式（移至设置界面）
- **视图模式** - 支持 safe/nsfw 模式切换，控制内容显示范围
- **侧边栏收起** - 可收起侧边栏，扩大内容区域
- **系统托盘** - 最小化到系统托盘，支持托盘菜单重启应用
- **页面记忆** - 重启后恢复上次打开的页面
- **模态框尺寸** - 统一模态框尺寸类（large/medium/small），优化界面一致性
- **统一导航** - 全屏查看器、图像详情、提示词编辑三个界面使用统一的 ListNavigator 类和导航按钮样式
- **禁止二级跳转** - 从提示词编辑界面进入图像详情页后，禁止再通过详情页按钮进入新的提示词编辑界面，反之亦然；禁用按钮置灰并显示"禁止二级跳转"提示

## 启动

### 生产环境启动

使用 `start-hidden.vbs` 启动，不显示命令行窗口：

```bash
start-hidden.vbs
```

### 调试模式启动

使用 `start-debug.bat` 启动，会显示控制台日志：

```bash
start-debug.bat
```

### 开发环境启动（需安装 Node.js）

在项目根目录执行：

```bash
# 安装依赖
cnpm install

# 启动应用
cnpm start
```

## 设置

设置界面采用卡片式布局，包含以下功能：

- **外观** - 切换明亮/暗黑主题
- **数据存储** - 更改数据存储目录
- **数据管理**
  - **导入** - 从 JSON 文件导入所有提示词数据
  - **导出** - 将所有提示词数据导出为 JSON 文件
  - **导出孤儿文件** - 导出不在数据库中的图像文件到指定目录并清理
- **危险操作**
  - **清空所有数据** - 一键清空所有提示词、图像和标签数据（不可恢复）, 保留图像文件

## 数据存储

默认存储在应用目录下的 `py-data/` 文件夹中，可在设置中更改：

- **SQLite 数据库**：`{数据目录}/prompt-manager.db`
- **图像文件**：`{数据目录}/images/`
- **缩略图**：`{数据目录}/thumbnails/`

### 数据库结构

使用 SQLite 数据库存储所有数据，主要表结构：

| 表名 | 说明 |
| ------ | ------ |
| `prompts` | 提示词数据（标题、内容、创建时间、更新时间、收藏状态等） |
| `images` | 图像数据（文件名、路径、MD5、尺寸、创建时间、更新时间、收藏状态等） |
| `prompt_tags` | 提示词标签 |
| `image_tags` | 图像标签 |
| `prompt_tag_relations` | 提示词与标签的关联关系 |
| `image_tag_relations` | 图像与标签的关联关系 |
| `prompt_image_relations` | 提示词与图像的关联关系 |
| `db_version` | 数据库版本记录 |

### 数据库迁移

使用版本号机制管理数据库结构变更：

- 自动创建 `db_version` 表记录已应用的迁移版本
- 启动时检查当前版本，只执行未应用的迁移脚本
- 支持增量更新，避免重复执行已完成的迁移

### 图像数据结构

```json
{
  "fileName": "原始文件名.jpg",
  "storedName": "时间戳_随机数.jpg",
  "path": "绝对路径",
  "relativePath": "images/存储名.jpg",
  "thumbnailPath": "thumbnails/thumb_存储名.jpg",
  "md5": "原图像MD5哈希",
  "thumbnailMD5": "缩略图MD5哈希",
  "width": 1920,
  "height": 1080,
  "createdAt": "创建时间",
  "updatedAt": "更新时间",
  "isFavorite": false
}
```

**MD5 去重**：上传图像时会计算 MD5，如果已存在相同 MD5 的图像，则复用已有文件，避免重复存储。

## 项目结构

### 必需文件/目录

| 路径 | 说明 |
| ------ | ------ |
| `main.js` | Electron 主进程入口 |
| `preload.js` | 预加载脚本，暴露安全 API |
| `database.js` | SQLite 数据库操作模块 |

| 路径 | 说明 |
| ------ | ------ |
| `{数据目录}/` | 数据存储目录（默认 `py-data/`，可配置） |
| `{数据目录}/prompt-manager.db` | SQLite 数据库 |
| `{数据目录}/images/` | 图像文件存储 |
